# 방송기술 관련 자료 번역

## Tektronix

[Tektronix](https://www.tek.com) 사의 기술 문서 번역들을 담고 있다. 각 문서 디렉토리 내의 out에 결과 pdf 파일이 있다.

### A Guide to Standard and High-Definition Digital Video Measurements
